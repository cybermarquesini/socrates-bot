
# Sócrates • Mentor do Tribunal

Este bot concede automaticamente o cargo "Discípulo da Razão" aos usuários que clicarem no botão após lerem o código de conduta.

## Arquivos principais
- `main.py`: código do bot
- `.env`: armazena seu token e IDs
- `requirements.txt`: bibliotecas necessárias

## Como rodar localmente
1. Instale o Python 3.10+
2. Crie um `.env` com base no `.env.example`
3. Instale as dependências: `pip install -r requirements.txt`
4. Rode: `python main.py`

## Para hospedar no Railway
1. Crie um projeto
2. Faça upload desses arquivos
3. Configure as variáveis de ambiente do `.env`
4. Clique em "Deploy"
